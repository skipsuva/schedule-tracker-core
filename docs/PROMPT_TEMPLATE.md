# Prompt Template

All meaningful work should start with a written prompt.

This creates:
- clarity
- replayability
- a durable paper trail

---

## Prompt: <Title>

### Objective
What is the single outcome this work should achieve?

---

### In Scope
- Bullet list of what is included

---

### Out of Scope
- Explicit exclusions

---

### Files Allowed to Change
- List files or directories

---

### Constraints
- Tech rules
- Architectural rules
- Performance or safety constraints

---

### Acceptance Criteria
- Clear, testable conditions
- “This is done when…”

---

### Notes
- Edge cases
- Open questions
- TODOs for future work
